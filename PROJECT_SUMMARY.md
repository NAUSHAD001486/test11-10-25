## 🔧 Quick Start

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Configure environment**:
   - Update `config.env` with your Cloudinary credentials
   - Update domain URLs in HTML files

3. **Test the application**:
   ```bash
   node test.js
   ```

4. **Start the server**:
   ```bash
   npm start
   ```

5. **Deploy to Hostinger**:
   ```bash
   ./deploy.sh
   ```
